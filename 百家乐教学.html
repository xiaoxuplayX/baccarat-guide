01📘 基础入门

百家乐基础规则与牌桌认识

了解百家乐的基本规则：庄家(B)与闲家(P)的补牌规则、点数计算方式、赔率结构。

  1. 点数计算：A=1, 2-9面值, 10/J/Q/K=0
  2. 只取个位数，最大9点
  3. 闲家5点以下补牌
  4. 庄家根据闲家第三张决定补牌
  5. 赔率：闲1:1, 庄1:0.95, 和1:8
02📘 基础入门

看路基础：大路的认识与记录方法

大路是百家乐最基本的路纸记录法，用红蓝圈记录庄闲结果。

  1. 红圈=庄赢, 蓝圈=闲赢
  2. 连续同结果向下排列
  3. 结果改变时换下一列
  4. 和局用绿线标记
  5. 对子用红/蓝点标记
03📘 基础入门

看路进阶：大眼路与小路的读法

大眼路和小路是从大路衍生的路纸，用来判断路型的规律性。

  1. 大眼路：比较当前列与前一列的走势
  2. 有路(规律)=红, 无路(不规律)=蓝
  3. 小路：比较当前列与前两列
  4. 同样用红蓝标记有路/无路
  5. 配合大路使用判断趋势
04📘 基础入门

看路技巧：曱甴路与珠盘路完整教学

曱甴路是第三条衍生路，珠盘路是最直观的逐局记录法。

  1. 曱甴路：比较当前列与前三列
  2. 规律性判断同大眼路逻辑
  3. 珠盘路：按顺序逐格记录
  4. 红=庄, 蓝=闲, 绿=和
  5. 6行排列，从左到右从上到下

  3. 负注法(负追)：输了加注
  4. 1-3-2-6法：经典胜进策略
  5. 关键：设总止损线和止盈线
